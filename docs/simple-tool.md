# `simple-tool`

单页面工具应用模板。

## 1 创建方式

```bash
magic-dash create --name simple-tool
```

## 2 应用初始化启动

- 安装项目依赖库

```bash
pip install -r requirements.txt
```

- 启动应用

```bash
python app.py
```

- 访问应用

应用默认地址：http://127.0.0.1:8050

## 3 项目目录结构

```bash
simple-tool
 ┣ app.py # 应用主文件
 ┗ requirements.txt # 项目依赖信息
```
